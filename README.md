# Курсовая 4 Сервис управления рассылками (Django)

## Функционал
Основные возможности:
-Клиенты - управление получателями (CRUD: создание, просмотр, редактирование, удаление)
-Сообщения - шаблоны писем с темами и текстами
-Рассылки - создание рассылок с указанием времени, выбором сообщения и клиентов
-Ручная отправка - запуск рассылки по требованию с кнопки в админке
-История отправок - запись всех попыток (успешных/неудачных)

## Открыть в браузере
Главная страница: http://127.0.0.1:8000/
